# plog
log parser written in Perl
# Usage
```
logParser [OPTIONS] [FILE] [PATTERN...]

OPTIONS:
   -h              show this help
   -o <file>       output file
   -D <dir>        fs tree with match
   -v              set verbose mode
FILE:
   file containing the logfilenames to analyse
PATTERN:
   can be a regex or a simple word to look for
```
